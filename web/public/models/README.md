Prototype 3D model drop-in notes
================================

Expected prototype model path:

`web/public/models/stitchra-shirt-prototype.glb`

Reference URL in code:

`/models/stitchra-shirt-prototype.glb`

Use only a locally downloaded model with a license that permits the intended
prototype use. Do not hotlink TurboSquid or any other marketplace files.

If the downloaded temporary asset is FBX, OBJ, or Blend, convert it to GLB with
Blender before using it on the website:

1. Open the asset in Blender.
2. Remove unused cameras, lights, animation tracks, and heavy background props.
3. Apply transforms so the shirt is centered at the origin and faces forward.
4. Reduce geometry and texture sizes enough for web use.
5. Export as glTF 2.0 Binary (`.glb`).
6. Save the exported file as `stitchra-shirt-prototype.glb` in this folder.
7. Set `available` to `true` in `stitchra-shirt-prototype.json`.

Temporary prototype 3D asset. Replace with original Stitchra 3D garment model
before final commercial launch.
